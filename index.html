온기(ON:GI)

‘5분의 기억 공감, 마음을 잇는 가족의 시간’

온기 서비스 바로 가기

온기는 무엇인가요?

  • 독창적이고 혁신적인 가족 중심 디지털 기억 공감·치유 플랫폼
  • “전문 연구기관이나 기업이 아닌, 인지 기능이 저하된 아버지를 위해 자녀들이 직접 만든 프로그램”
  • 사람 중심·가족 중심 혁신: 마음과 기억을 연결하는 새로운 효도의 기준 제시
  • 짧은 5분의 기억 회상을 통한 부모님의 인지 기능 활성화, 정서 안정, 가족간의 교감과 소통 강화

온기는 왜 필요한가요?

  • 부모님·가족: 기억 회상 및 정서 교감을 통해 인지 기능 향상과 정서 안정
  • 노인 보호기관·복지시설: 기억 회상 데이터를 기반으로 맞춤형 케어 제공 → 운영 효율성 향상, 공공 지원 연계
  • 국가기관·지자체: 데이터 기반 정책 설계와 ESG·공익 연계 → 디지털 복지 정책 실현

온기의 핵심 가치

  • 짧은 시간, 깊은 교감 — 5분 만에 마음과 기억 연결, 인지 자극·기억력 향상
  • 가족 맞춤형 경험 — 사진·영상 기반 회상, 자연스러운 대화 유도
  • 누구나 참여 가능 — 고령층도 학습 없이 쉽게 사용
  • 정서 안정과 행복감 — 연구 결과: 우울·불안 감소, 자존감·인지 기능 향상
  • 개인별 기록 활용 — ‘기억 리포트’로 치유·복지 연구에 활용 가능
  • 공익적 보급 — 전문가 중심이 아닌 가족 경험 중심 설계

기관·시설 활용

노인 복지시설, 주·야간보호센터 등에서 단체 프로그램으로 운영 가능.
기억 회상 세션, 사진·영상 기반 퀴즈, 반응 기록, 리포트 생성, 통계 제공.

사회·경제적 가치

  • 부모님·가족: 기억 회상 → 인지 기능 향상, 정서 안정, 세대 간 교감 강화
  • 기관·시설: 맞춤형 관리, 운영 효율성 향상
  • 국가기관·지자체: ESG·공익 연계, 복지 모델 구축
  • 사회적 파급: 복지 비용 절감, 연구·정책 근거 제공
